Overview
********

This section was built in order to lead new developers how to understand the
kytos project. First of all you must understand the Kytos architecture reading
the :doc:`Admin Guide </administrator/architecture>`. After that you may learn
how to install, configure and understand the develop environment trying to do
the `tutorials <http://tutorials.kytos.io/>`__ made by Kytos Team.
