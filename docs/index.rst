###################
Kytos Documentation
###################

Kytos documentation has two main parts:

Administrator Guide
  For system administrators to install Kytos Platform and manage Network
  Applications (NApps)

Developer Guide
  For those who want to create NApps and/or contribute to Kytos SDN platform
  source code. You should read the Administrator Guide as an introduction
  before learning more details here.

Other parts include information on how to get help, get involved, authors and
license.
